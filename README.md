# Cat Blog

This website is a school project created by BSIT4C Students from Bulacan State University Hagonoy Campus.

## Description

Welcome to the Cat Blog! This project is a space dedicated to all things feline, where you can explore articles, pictures, and fun facts about our furry friends.

## Purpose

The main purpose of this website is to serve as a learning endeavor for our academic studies. Through this project, we aim to apply the skills and knowledge acquired during our studies in BSIT4C at Bulacan State University Hagonoy Campus.

## Features

- Browse through a collection of articles and blog posts about various cat breeds.
- Enjoy a gallery of adorable cat pictures.
- Learn interesting facts about cats' behavior, care, and history.

## Technologies Used

- HTML
- CSS
- JavaScript
- Python
- Django Framework
- Bootstrap 5

## How to Use

0. Make sure that `Python` and `Django` are installed.
1. Clone this repository.
2. Open the clone folder
3. In the terminal, install all requirements from requirements.txt using pip
```
pip install -r requirements.txt
```
4. Go to the project main directory via `cd`.
```
cd CatBlog
```
5. Prepare the models and tables via migrations.
```
py manage.py makemigrations
```
and then..
```
py manage.py migrate
```
6. Run Django servers to start the website.
```
py manage.py runserver
```

Happy exploring! 😺🐾
